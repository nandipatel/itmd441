ITMD 441 - Web Web Applications Foundation

Course Outcomes:

Students completing this course learn about the core technologies used to develop contemporary web applications.

This includes the primary protocols, tools, software, markup, and scripting languages used in modern web development and how they have progressed over time.

Each student successfully completing this course will demonstrate a strong foundational knowledge in the design and development of web-based internet applications.

Students will use standards compliant HTML and CSS to create responsive user interfaces targeting modern browsers.

Additionally, students learn about the JavaScript language, debugging techniques, and JavaScript APIs and how to effectively utilize them.
